The User Management module is responsible for managing user accounts and access permissions within the Back Office system. Administrative users are created from employee records and assigned permissions according to their responsibilities. Since the system is used across multiple departments, permission control is important to ensure that users can only access authorized functions. The system supports role-based access control through user groups and permission assignments. For example, Human Resource staff can manage attendance-related information, while other departments may only have view permissions. Managers are granted access to team-related functions, while the superuser has full access to all system functions and system configuration settings. This module helps ensure system security and proper separation of responsibilities across the organization.

The Company Structure Management module was developed to improve the management of organizational information within the system. Since attendance records, reporting, permissions, and employee assignments depend on organizational data, a centralized structure was required to maintain consistency across all modules.

The module manages companies, branches, departments, teams, positions, and manager assignments. It allows administrators to define organizational relationships and maintain employee assignments according to the company's operational structure.

The company structure information is used throughout the system for attendance processing, report filtering, shift planning, and access control. By maintaining a well-defined organizational hierarchy, the system can provide more accurate reporting and improve data organization for Human Resource operations.

The Leave record module was developed to provide a centralized solution for managing employee leave records. Previously, attendance processing relied primarily on attendance records without comprehensive integration of leave information. The module allows administrators to manage leave types, leave balances, leave records, and leave summaries. Employees can be assigned different leave categories according to company policies, while Human Resource staff can monitor leave usage and available balances. Additional features include half-day leave support, leave balance calculation, leave history tracking, and attendance integration. Approved leave records automatically affect attendance status calculations, helping ensure that employee absences are correctly classified. This module improves attendance accuracy and simplifies leave administration activities.
